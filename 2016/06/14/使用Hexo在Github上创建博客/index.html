使用Hexo在Github上创建博客

安装

环境配置

  • 安装Node.js
  • 安装git
  • 创建github账号

安装Hexo

输入：
npm install -g hexo-cli

初始化博客文件夹

创建一个文件夹，如blog，然后：

cd blog
hexo init
npm install

在这之后，可以通过本地服务器查看博客网站的内容：

hexo server

根据hexo提示的网址从浏览器进入，默认会增加一篇hello world，介绍了hexo的基本用法。

基本命令

新建文章

hexo new article-title

创建新文章，article-title为文章标题，如果标题中有空格，用””双引号括住标题。

创建之后，可执行以下命令：

hexo generate // 生成静态网页以及css、js文件，存放于public目录中
hexo server   // 本地服务器查看
hexo deploy   // 部署到远端服务器，配置github之前不要使用

Github配置

Github Repository

创建一个repository，名称应该是github-username.github.io，其中github-username是你的github账号名。

Hexo配置

在能够部署到github之前，需要安装插件：

npm install hexo-deployer-git --save

前面创建的Hexo博客目录中，有一个_config.yml文件，需要对其进行编辑：

deploy:
  type: git
  respository: https://github.com/github-username/github-username.github.io.git
  branch: master

Hexo目前支持的类型是git，不再是github，分支设置为master。这样配置之后，上传完成以后，你的网站就可以通过http://github-username.github.io进行访问了。

同时要修改一下_config.yml文件中的url部分，改为你的网站网址。

其它配置可以参阅Hexo配置文档。

每次创建网页之后，执行：
hexo generate
hexo deploy

完成必要的生成和发布。
